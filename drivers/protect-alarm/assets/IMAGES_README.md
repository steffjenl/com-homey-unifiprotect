# Image Assets Required

The protect-alarm driver requires two PNG images:

1. **large.png** - 500x500px icon for device tiles
2. **small.png** - 75x75px icon for device lists

## Recommended Design

Since this is an alarm manager, consider using:
- Shield icon with checkmark (armed)
- Bell or siren symbol
- Home security iconography
- UniFi color scheme (#159EDA brand color)
