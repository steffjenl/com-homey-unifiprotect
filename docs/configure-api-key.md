# 🔐 Creating an API Key for UniFi Protect

Follow these steps to generate and configure an API key for integration with the Homey UniFi Protect App.

---

## 1. Access UniFi Protect
- Navigate to your **local UniFi environment**.
- Open the **Protect** application.

## 2. Generate the API Key
- Click the **gear icon** (⚙️) to open settings.
- Go to **Control Plane**.
- Select **Integrations**.
- Enter a name for the API key (e.g., `Homey Protect Integration`).
- Click **Create API Key**.
- **Copy** the newly generated API key.

## 3. Configure Homey UniFi Protect App
- Open the **Homey UniFi Protect App settings**.
- Paste the **API key** into the designated field.
- Click **Save** to apply the settings.

---

✅ Your Homey app is now connected to UniFi Protect using the API key.
